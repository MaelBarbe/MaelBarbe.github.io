Widget Twitch Personnalisé

Un système de gestion des assets visuels personnalisés pour enrichir l'apparence des streams Twitch.

Ce projet personnel est un widget Twitch personnalisé conçu pour transformer l'aspect visuel des streams. Il propose un système de gestion d'assets graphiques créés sur mesure, permettant une meilleure expression visuelle et une image de marque cohérente dans l'univers Twitch.

Caractéristiques principales

  • Galerie d'assets organisée : Collection d'images PNG représentant différentes états et émotions (idle, happy, angry, shy, wink, confused) facilement accessibles.
  • Assets hébergés sur GitHub Pages : Gestion centralisée de tous les fichiers visuels pour un accès rapide et une maintenabilité optimale.
  • Interface de gestion intuitive : Page dédiée pour visualiser et copier facilement les URLs de tous les assets du widget.

Outils

HTML, CSS, JavaScript

Hébergement

GitHub Pages

Type

Projet personnel

Ce que j'ai appris

Ce projet m'a permis de maîtriser plusieurs aspects du développement web :

  • Gestion et organisation efficace d'une galerie d'assets visuels
  • Optimisation des performances pour le chargement d'images multiples
  • Déploiement et gestion de projets avec GitHub Pages
  • Création d'une interface fonctionnelle pour gérer les ressources
