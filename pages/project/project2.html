Script Python pour renommer un grand nombre d'images

Lors d'un projet pour une photographe freelance durant lequel j'ai réalisé le design et l'intégration de son site web, j'ai eu le besoin de renommer un grand nombre d'images pour qu'elles aient un nommage homogène. J'ai donc décidé de créer un script Python qui permet de renommer automatiquement les images. J'ai décidé de le créer parce que j'avais également envie de découvrir le langage avec un petit projet.
